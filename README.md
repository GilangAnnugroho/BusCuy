**Bus Cuy** adalah aplikasi mobile berbasis Flutter yang dirancang untuk memudahkan penyewaan bus pariwisata. Aplikasi ini membantu pengguna mencari armada bus, melihat fasilitas, dan melakukan pemesanan (booking) secara langsung melalui smartphone.

Project ini dikembangkan sebagai bagian dari tugas akhir/project kuliah Teknik Informatika.

ğŸ“– Panduan Penggunaan (User Guide)
Berikut adalah alur penggunaan aplikasi Bus Cuy:

### 1. Registrasi & Login
Buka aplikasi, jika belum punya akun silakan klik menu Daftar.
Masukkan Nama, Email, dan Password.
Setelah berhasil, login menggunakan akun tersebut.

### 2. Mencari Bus
Di halaman Home, Anda akan melihat daftar bus rekomendasi.
Gunakan fitur Search di bagian atas untuk mencari bus berdasarkan nama.

### 3. Melakukan Pemesanan
Pilih salah satu bus untuk melihat Detail.
Klik tombol Sewa Sekarang.
Tentukan Tanggal Mulai dan Tanggal Selesai.
Klik Konfirmasi untuk mengirim pesanan.

### 4. Cek Status
Masuk ke menu Riwayat di navigasi bawah.
Anda dapat melihat tiket yang statusnya masih Pending atau sudah Dikonfirmasi.
